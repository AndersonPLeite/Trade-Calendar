# 📈 Calendário de PnL para Trading

Um diário de trading profissional desenvolvido para **day traders** acompanharem desempenho diário, lucro e prejuízo (PnL) e consistência operacional.

A aplicação permite registrar resultados diários e visualizar a performance através de um calendário interativo focado exclusivamente em **dias de mercado**.

---

## 🚀 Funcionalidades

✅ Registro diário de trades
✅ Controle de Lucro e Prejuízo (PnL)
✅ Calendário apenas com dias de negociação (sem finais de semana)
✅ Visualização de performance mensal
✅ Persistência local dos dados
✅ Interface interativa baseada em calendário

---

## 🧠 Motivação

Muitos traders falham não por estratégia, mas por falta de acompanhamento e disciplina.

Este projeto foi criado para simular um **diário de trading profissional**, semelhante às ferramentas utilizadas por **prop firms** e traders quantitativos.

## 📦 Instalação

Clone o repositório:

```bash
git clone https://github.com/AndersonPLeite/trade-calendar.git
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
